# Be My Valentine

Une page web romantique et ludique pour une demande de Saint-Valentin, avec un bouton « Non » qui se déplace et une animation de confettis en forme de cœurs quand la réponse est « Oui ».

## Fonctionnalités

- Interface pastel et typographie romantique.
- Bouton « Non » insaisissable sur ordinateur et mobile.
- Animation de cœurs lors du clic sur « Oui ».
- Mise en page réactive.

## Démarrage

Ouvrez simplement `index.html` dans votre navigateur.

## Personnalisation rapide

- Modifiez le texte dans `index.html` pour adapter le message.
- Remplacez l’image `4f41f439-6bda-4481-b5df-2184142c9f9c.png` par votre propre illustration (gardez le même nom ou changez la référence dans `index.html`).

## Structure

- `index.html` : page principale.
- `4f41f439-6bda-4481-b5df-2184142c9f9c.png` : bannière visuelle.
